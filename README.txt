سیستم جامع گواهینامه و امتیازدهی کسب و کارها

== Description ==

HosNo Certificate یک سیستم کامل برای مدیریت، ارزیابی و گواهینامه کسب و کارها است.

== Features ==
* ثبت و مدیریت کسب و کارها
* سیستم امتیازدهی پیشرفته
* مقایسه کسب و کارها
* رزرو آنلاین
* داشبورد اختصاصی

== Installation ==
1. افزونه را آپلود کنید
2. فعال کنید
3. تنظیمات را پیکربندی کنید

== Frequently Asked Questions ==
